SingleThreadedEventQueue

the EsEventQueue is a single threaded thread pool which is used to act as the only thread using an instance of the spidermonkey runtime besides being able to add tasks from any thread by add_task a running task can add jobs to the current thread by calling add_task_from_worker those tasks need not impl the Send trait and there is no locking happening to add the task to the queue
